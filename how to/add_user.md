## Add user to zabbix

1. Navigate to User Managment 
Go to: <br>
Administration → Users → Add user (top-right button)<br>
You will see three tabs: User, Media, and Permissions.<br>

- User:
 1. username:ersilda
 2. group:internal (or any other appropriate group)
 3. password: (at least 8 characters)
- Media:
 1. Gmail
  - Click Add
  - Select Gmail as the media type
  - Enter the user’s email addres
  - Set the active time period
  - Enable all severity levels (recommended for testing)
 2. Telegram
  - Click Add
  - Select Telegram as the media type
  - Enter the Telegram Chat ID
  - Set operating hours and select the severity levels
  >Other media types can be added as well
- Permissions:
 For testing purposes, select the Admin role. Roles and their permissions can be customized at: Administration → User roles
- Click Update and now the new user is set.
