## Modify MAP
1.  Navigate to Maps
Go to: Monitoring â†’ Maps

2. Select the Map you need to modify
Click on the map name you want to modify.
At the top-right corner, click:
Edit map
This opens the Zabbix Map Editor.
3. Add a host
- Click Add element
- Click anywhere on the map canvas to place it.
- Configure it as Host
- Choose Label
- Choose the specific host
- Set an Icon and the size
- Add host to map
- Add link to zabbix server or other devices it might be connected. For wireless connections represent it with a dashed line.
