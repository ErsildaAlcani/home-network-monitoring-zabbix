# monitoring-home-network-zabbix
### Zabbix Server Installation on Ubuntu 24.04 ARM64 (UTM on macOS M2)
This project demonstrates a **home network monitoring setup using Zabbix 7.0 LTS** on an **Ubuntu Server 24.04 ARM64** virtual machine using **UTM** on a **MacBook Air M2**. The VM is configured to be accessible on the local network.
It showcases how to monitor PCs, servers, and network devices in a home environment.  
The goal is to track uptime, resource usage, and service availability.

---

## Features
- Monitor multiple devices (Windows/Linux/MacOS) on a LAN
- Track CPU, memory, disk, and network usage
- Real-time dashboards and historical data
- Example agent configuration for monitored hosts
- Alerts and email notifications

## What can be done?

- Monitor a network device using zabbix agent.
- Monitor a windows network device using SNMP service.
- Monitor a virtual machine host using zabbix agent.
- Configure Map and Dashboard for easier access.
- Add a virtual machine host.
